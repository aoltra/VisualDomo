VisualDomo

Versión: 0.4.2 (alfa)
Fecha: Dec 2014
Autor: Alfredo Oltra

VisualDomo allows a visual control of the ODControl devices from OpenDomo Services SL company.

VisualDomo is written using HTML, CSS and JavaScript with the support of several frameworks like jQuery, jQuery Mobile and Cordova (also known as PhoneGap).
